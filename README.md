# PC4 - Control Estadístico de Procesos (Amazon Robotics)

**Desarrollado por:** Milagros Lidia Mayta Soto  

Este proyecto simula una evaluación técnica empresarial donde se controla el peso de sensores de la empresa Amazon Robotics mediante gráficos de control de calidad.

## Estructura del Entregable
- `index.html`: Dashboard interactivo premium con la portada, enunciado, identificación de la carta, desarrollo en Excel y Minitab, interpretación y conclusiones de ingeniería.
- `excel/solucion.xlsx`: Plantilla oficial de Excel con cálculos dinámicos de medias, rangos y límites de control.
- `minitab/`: Carpeta destinada a guardar el proyecto Minitab (.mpx).
- `prompts/prompts.md`: Historial y registro de interacciones con IA.
- `declaracionIA.md`: Declaración jurada de autoría y porcentaje de apoyo tecnológico.

## Ejecución y Visualización
1. Para ver el dashboard interactivo, abra el archivo `index.html` en cualquier navegador web moderno.
2. Para revisar los cálculos, abra `excel/solucion.xlsx` en Microsoft Excel. Las fórmulas se recalcularán automáticamente si se alteran los datos de entrada.
